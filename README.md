## 1. 리눅스 명령어

리눅스 환경에서는 좀 더 효율적인 작업을 위해 명령어를 사용한다. 좀 더 다채롭게 활용할 수 있는 명령어는 다음과 같다.

## **top**
- 리눅스 실행 이후 부터 메모리 사용량, 현재 작업 프로세스, cpu 사용량 등을 알려주는 명령어이다. 이 명령어를 사용하면 시스템에 로그인 된 사용자의 수와 시스템 부하율도 확인이 가능하다. 현 상태를 한 눈에 확인 가능하다는 장점이 있다.
